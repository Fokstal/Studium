# Для запуска проекта:

## Установить:
    - Docker;
    - Docker-Compose;


## Запуск:
    1. Открыть корневую папку проекта в терминале.
    2. Запустить проект с помощью команды `docker-compose up`.
    3. Дождаться установки всех пакетов и зависимостей.
    4. Перейти по адресу "http:localhost:3000".
